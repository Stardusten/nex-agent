  - 冻结新的 app 心智：Workbench runtime 只服务静态 artifact；app source layout 归 app 自己所有。
  - 标准 app 目录包含 `nex.app.json`、静态 entry/assets，以及可选 `reload.sh`、`src/**`、`package.json`、`dist/**`。
  - simple app 可直接改 `index.html` / `app.js` / `style.css` 后刷新 iframe；buildable app 用 app-local `reload.sh` materialize `dist/index.html` 等运行产物。
  - `reload.sh` 是受控 app artifact refresh hook，不是 core CODE deploy、browser iframe reload、arbitrary shell 或 permission approval。
